# Sova

Sova is a functional programming language inspired by Lisp, Python and JavaScript

## Installation

To install globally

```
npm install -g sova
```

## Usage

To run `.sv` file

```
sova index.sv
```

## Syntax

### Declaration

To declare named value

```
= name value
```

Examples:

```
= integer 1
= float 1.4
= string 'Hello'
= boolean true
```
