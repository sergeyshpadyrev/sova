# Sova

<i>Sova</i> is a functional programming language inspired by Lisp, Python and JavaScript

## Features
- Lisp-like syntax
- Python-like indentation structure
- Transpiling to JavaScript

## Showcase
```
= a 1
= b 2
 
console.log
  ? (> a b) 'Greater' 'Less'
```
Result output: `Less`
